# YENİDEN - Restoran Menü ve Yönetim Sistemi

Müşteri odaklı dijital menü ve dinamik ürün yönetim sistemidir.

## Özellikler
- **Müşteri Paneli:** Kategorize edilmiş ürünler, mobil uyumlu şık arayüz.
- **Yönetim Paneli:** Admin paneli üzerinden anlık ürün ekleme, silme ve düzenleme.
- **Güvenlik:** Şifre korumalı admin girişi.

## Teknolojiler
- Python (Flask)
- PostgreSQL
- HTML/CSS (Responsive Tasarım)

## Kullanım
- QR kod üzerinden menüye erişim.
- `/admin` üzerinden yönetim paneline erişim.
